We are Nitrolabs

At Nitrolabs we work with the open-source community to develop new libraries, apps and software that push the boundaries of computing technology. We believe in a world where talented software developers work together to create amazing products an services.

About Nitrolabs

Write great code, build awesome apps, get rewarded.

Writing code is fun! But with large corporate companies dominating the technology market, it is difficult to make a change by yourself. Nitrolabs is a collaborative software development group. People from all over the world help build and maintain the Nitrolabs code base. Anyone can help us develop amazing new apps.

  • Developed by the community
  • Open-source where possible
  • Build with the latest technology

We are currently trialling a system where programmers and designers alike get rewarded for contributing to Nitrolabs projects. Essentially, we divide the ownership of each project among the programmers and designers who built it. Users are scored based on their activity and contribution to each project. The entire scoring process is fully automated, but also incorporates survey data from each person in the team.

Sound Interesting? Have a look at our Github Account, and send us a message on Github.

Projects We Support

Meteor

Meteor is a complete open source platform for building web and mobile apps in pure JavaScript. Read about the Meteor framework on their project website

FileFog

A thin cloud-service agnostic interface to access and execute file operations in Node Allows Google Drive, Dropbox, OneDrive and others to be accessed from a single API

Facebook Collections

Popular package for Meteor that exposes the facebook-graph as a Meteor collection.

Node

Node.js is a platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications. If you have not tried it already, check out the node!

Django

Django is a high-level Python Web framework that encourages rapid development and clean, pragmatic design. Built by experienced developers, it takes care of much of the hassle of Web development.

Meteor Authentication

Nitrolabs maintains the Meteor authentication packages for Dropbox, Box, and OneDrive. These packages allow quick and easy support for three of the largest cloud providers
